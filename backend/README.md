# Backend Service (NestJS)

This directory contains the server-side logic for the AI English Scorer.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```

> **Note**: For full project documentation, environment setup, and architecture details, please refer to the **Root README** (`../README.md`).
